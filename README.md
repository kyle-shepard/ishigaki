# Ishigaki (石垣)

A persistent, multiplayer web game where players grow a single tiny hamlet — over
real, slow time — into a village, a castle town, a domain, and eventually a realm.

The design north star is [Lands of Lords](https://www.landsoflords.com/): a
browser-based, map-driven, feudal MMO with a deliberately slow real-time economy and
player-run politics. The initial goal is a close clone of that feel, which we then
diverge from and make our own.

The twist is setting: **feudal Japan** rather than medieval Europe. *Ishigaki* — the
fitted-stone base of a Japanese castle — is what everything is built on top of.
